IObit Uninstaller

IObit Uninstaller — это удобная бесплатная программа для полного удаления ненужных программ, папок, панелей инструментов и плагинов браузеровот компании IObit. Удаляет файлы как с жёсткого диска, так и с реестра системы. Не оставляет следов, используя "Мощное сканирование" и "Принудительное удаление". Эта эффективная программа для полного удаления ненужных программ, браузерных расширений и встроенных приложений Windows. Приложение умеет автоматически удалять остатки ненужных программ после их удаления, поддерживает все популярные браузеры (Chrome, Firefox, IE, Opera) и позволяет быстро находить и удалять вредоносные панели инструментов. Кроме этого IObit Uninstaller дает возможность обновлять установленное ПО и содержать его в актуальном состоянии (присутствует поддержка более 60 критических важных программ).

Функции:

  • Создание точек восстановления системы
  • Выявление программ, редко используемых пользователем
  • Удаление ненужных программ
  • Возможность пакетного удаления ненужных программ
  • Возможность восстановления удалённой программы
  • Опции «глубокая проверка» и «принудительное удаление» позволяют полностью удалять любые панели инструментов
  • Принудительное удаление для тех приложений, которые не могут быть удалены через собственные деинсталляторы
  • Программа не требует инсталляции, но требует деинсталляции, поскольку после запуска без спроса интегрируется в систему

Программа существует в бесплатной Free и платной Pro версиях. Последняя отличается большим функционалом.

Пример работы бесплатной версии программы на ОС Windows 7.

При запуске программы открывается вкладка Все программы, где находится информация обо всех программах, установленных на компьютере.

Удалим какую-нибудь программу, чтобы привести пример работы IObit Uninstaller'a:

Выберем программу. Сразу же видно количество памяти, которая будет освобождено.

Нажмем кнопку Деинсталлировать или иконку корзины напротив программы. Всплывает следующие окно:

Здесь пользователь может выбрать, какое действие предпринять перед удалением: Создать точку восстановления системы (полезно, когда не знаешь, что произойдет с системой после удаления) и Автамотически удалять все остаточные файлы (чтобы удалить подчистую). Выберем сразу два предложенных варианта и нажимаем Деинсталлировать.

Процесс удаления:

После этого выскочило уведомление об успешном удалении программы. Однако, ярлыки удаленной программы на Рабочем столе остались, хотя я выбирал Автамотически удалять все остаточные файлы.

Помимо окна Все программы в меню слева есть еще несколько вкладкок. Разберем их ниже.

В окне Пакеты ПО находятся различные версии установленных на ПК программы. На данном ПК пакетов нет:

Окно Недавно установленные - это сортировка всех программ по дате установки.

Окно Большое программы содержит список программ, свыше определенного объема занимаемой памяти.

Окно Редко используемые хранит в себе список программ, которые, очевидно из названия, редко используются. Удобно, когда программ на компьютере много, и какие-то не используются, поэтому их можно удалить за ненадобностью.

Окно Мониторинг установки позволяет добавить(автоматически в Pro версии) установочный файл и отследить изменения в системе во время установки ПО. Благодаря этому полная очистка во время деинсталляции произойдет более качественно.

В окне Обновление ПО находиться информация об обновлении версий программ, установленных на компьютере.

Видно, что почему-то только одна программа находиться в этом списке.

В окне Панель инструментов сореджит в себе список всех плагинов и расширений для браузеров. Можно посмотреть конкретно для каждого браузера. Для этого следует выбрать интересующий браузер, а не Все Панели инструментов.

Окно Центр действий содержит список ПО для оптимизация ПК различной направленности. По большому счету перечень программ компании IObit.

В правом верхнем углу, помимо стандартных кнопок Свернуть, Развернуть и Закрыть, располагаются кнопки Инструменты и Опции.

Нажам на вкладку Инструменты, выскакивает окно с шестью различными функциями.

Инструмент Здоровье ПО проводит анализ программ:

Инструмент Простая деинсталляция представляет собой прицел на Рабочем столе, на который можно перетащить какой-либо файл, или же сам прицел перетянуть на какой-то файл для удаления. При наведении появляется такое меню:

Инструмент Принудительная Деинсталляция позволяет удалить ПО, которое не хочет удаляться простым способом. При нажатии открывается такое окно, где можно выбрать программу для удаления.

В интрументе История удаления находится информация обо всех удаленных за последнее время программах.

Инструмент Уничтожитель файлов позволяет удалить файлы и папки без возможности на их восстановление.

Инструмент Обновления Windows дает возможность удалить ненужные обновления системы для освобождения места.
